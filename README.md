## Project setup

Before you start please add envroiment variables:

- VUE_APP_IP_URL=https://ipinfo.io/
- VUE_APP_IP_KEY - [Get API KEY](https://ipinfo.io/account/token)
- VUE_APP_MAP_API_KEY - Google Maps JavaScript API KEY

```
npm install
```
